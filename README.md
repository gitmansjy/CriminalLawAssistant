# 刑法知识问答助手 ⚖️

基于 LangChain + Streamlit + 通义千问 的智能刑法问答系统。

## ✨ 功能特点

- 📚 **多格式文档支持**：支持 TXT、PDF、DOCX、DOC、Excel、CSV 等格式
- 🔍 **智能检索**：使用向量数据库 + MMR 搜索
- 🤖 **问题理解**：使用大模型改写问题，提升检索准确率
- ⚖️ **法律专用**：针对刑法、刑事诉讼法、司法解释优化
- 📊 **知识库管理**：自动分类统计法律文件
- 💬 **对话式交互**：友好的聊天界面

## 🛠️ 技术栈

- **前端框架**：Streamlit
- **LLM框架**：LangChain
- **向量数据库**：ChromaDB
- **Embedding模型**：all-MiniLM-L6-v2
- **大模型**：通义千问 qwen-plus
- **文档处理**：PyPDF、python-docx、unstructured
